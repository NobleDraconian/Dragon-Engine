DragonEngineClient

This item only works when running on the client. Client

Handles the client sided aspects of the framework such as controllers, connecting to remotes, etc.

Types​

Controller​

</>
interface Controller {
Init: function--

The controller's Init method.

Start: function--

The controller's Start method.

...: function--

The controller's various defined methods.

}

A client-sided microservice.

ClientPaths​

</>
interface ClientPaths {
ModulePaths: table--

The folders that the framework will lazyload modules from

ControllerPaths: table--

The folders that the framework will load controllers from

}

The folders that the framework will look in when attempting to load controllers & modules. Here's an example of a valid ClientPaths configuration: 

{
	ModulePaths = {
		Client = {
			Players.LocalPlayer.PlayerScripts.Modules
		},
		Shared = {
			ReplicatedStorage.Modules,
		}
	},
	ControllerPaths = {
		Players.LocalPlayer.PlayerScripts.Controllers
	}
}

For more information, see client paths.

Functions​

GetService​

</>
DragonEngineClient:GetService(
ServiceName: string--

The name of the service to get a reference to

) → ServiceClient--

The service with the given name

Returns a reference to the requested service

local MarketService = DragonEngine:GetService("MarketService")
MarketService:RequestPurchase("HealthPotion",5)

GetController​

</>
DragonEngineClient:GetController(
ControllerName: string--

The name of the controller to get a reference to

) → Controller--

The controller with the given name

Returns a reference to the requested controller

local UIController = DragonEngine:GetController("UIController")
UIController:DisplayDialogue("Do you like Cake, or Pie?","Cake",Pie")

RegisterControllerClientEvent​

</>
DragonEngineClient:RegisterControllerClientEvent(
Name: string--

The name to assign to the BindableEvent

) → BindableEvent--

The BindableEvent that was registered with the framework

Registers a BindableEvent for the calling controller that it can use to fire client-side events.

local AvatarJumpedBindable = DragonEngine:RegisterControllerClientEvent("AvatarJumped")
AvatarJumpedBindable:Fire(PlayerCharacter)
warning

This API should only be called from a controller! Calling it outside of a controller will cause errors.

Run​

</>
DragonEngineClient:Run(
FrameworkSettings: FrameworkSettings--

The settings to configure the framework's behavior

) → nil

Runs the framework. Controllers will be loaded & ran from the service locations specified in the framework's settings, and modules will be marked for lazyloading from the locations specified in the framework's settings. 

DragonEngine:Run()
warning

This API should only be called once! Calling it more than once will result in unstable behavior.

warning

If the framework is accessed before this API is ran, nothing will have been initialized!
