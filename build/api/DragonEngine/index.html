DragonEngine

The core of the framework. It handles module-loading, logging, and other aspects that are available on the server and client.

Types​

LogType​

</>
type LogType = string

The type of a log. Valid values are Normal, Warning and Error.

FrameworkLog​

</>
interface FrameworkLog {
Message: string--

The log's message

Type: LogType--

The log's type

Timestamp: string--

The timestamp of when the log was created

}

A table containing the metadata of a single log.

FrameworkSettings​

</>
interface FrameworkSettings {
ShowLogoInOutput: bool--

Determines whether or not the dragon engine logo is shown in the output when the framework runs.

Debug: bool--

Determines whether or not any debug logs logged via DragonEngine:DebugLog() will be displayed.

ServerPaths: ServerPaths
ClientPaths: ClientPaths
}

The general settings of the framework. For more information, see framework configuration.

Functions​

GetOutput​

</>
DragonEngine:GetOutput(
...: any--

The value(s) for the framework to return from this API call.

) → string--

The output the framework returns in response to this API call.

Returns output from the framework. Useful for checking if the framework is being accessed properly.

print(DragonEngine:GetOutput("HelloWorld",os.clock(),true))

Log​

</>
DragonEngine:Log(
LogMessage: string,--

The message to add to the logs

LogMessageType: LogType--

The type of the message that is being logged.

) → nil

Adds the given text to the framework's logs.

DragonEngine:Log("EquipPet() was called, but the player has no avatar!","Warning")

DebugLog​

</>
DragonEngine:DebugLog(
LogMessage: string,--

The message to add to the logs

LogMessageType: LogType--

The type of the message that is being logged.

) → nil

Adds the given text to the framework's logs if FrameworkSettings.Debug is true.

DragonEngine:DebugLog(
	("The item '%s' was purchased by player '%s'"):format(ItemName,Player.Name),
	"Normal"
)

GetLogHistory​

</>
DragonEngine:GetLogHistory(
MaxLines: integer--

How many lines back of history to return. If omitted, all logs will be returned.

) → {FrameworkLog}--

The logs pulled from the log history, in ascending order

Returns the history of the output logs, in ascending order (older entries first).

print(DragonEngine:GetLogHistory(50)[4].Message)

GetModule​

</>
DragonEngine:GetModule(
ModuleName: string--

The name of the module to get from the framework

) → table--

The module with the given name

Gets and returns the specified module from the framework if it exists. Returns nil if it does not exist. If this is the first time the module is being called, it will be lazyloaded. 

local AvatarUtils = DragonEngine:GetModule("AvatarUtilities")
AvatarUtils:CreateCharacterModelFromPlayer(Player)
