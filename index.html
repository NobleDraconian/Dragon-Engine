Dragon Engine

Opensourced and powerful framework for your Roblox games.

Learn more

What is Dragon Engine?

Dragon Engine is a Lua framework designed specificially for Roblox. It bridges the gap between the server and client, globally loads modules to allow for easy communication (this also helps to prevent Cyclic requiring), and is designed to serve as the 'backbone' of the game it is in.

Modular

Almost every part of the framework is modular. This means you can easily take segments of the framework that you need and deploy them in another game.

Package friendly

The framework is 100% compatible with packages. This makes it extremely easy to update the framework and push changes across massive multi-place universe games. This also has the added benefit of adding version control, so you can revert framework updates if needed!
